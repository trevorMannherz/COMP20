I used the Cat facts API which gives a bunch of cat facts and also lets you send facts to your friends and look at user data so long as you are logged in on your account.

I found it on this website: https://alexwohlbruck.github.io/cat-facts/?ref=apilist.fun

I used the facts option on the API request which just returns cat fact info.

This API could have a use on a website for kids where you want to provide random fun facts about animals, and the one for cats specifically could pull from this API.

This API could also be used on a website which sells cat products and could have a section with cat fun fact of the day which pulls from this API.
